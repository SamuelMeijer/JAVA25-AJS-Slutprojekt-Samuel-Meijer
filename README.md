# AJS - Scrum Board

## Instructions:
1. Clone the repo
2. In the root-directory open a terminal and write:
  npm install
3. Once everything is installed write in the root-directory:
  npm run build
4. If any error occurs:
  4.1 Navigate to the /frontend-directory and write:
      npm run build
  4.2 Navigate to the /backend-direcotry and write:
      npm run build
5. If you want to start the app at a later stage, navigate to the root-directory and write:
      npm run start
   5.1 Or if that isnt working, navigate to the /backend-directory and write:
      npm run start
